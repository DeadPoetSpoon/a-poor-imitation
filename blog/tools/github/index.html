GitHub Actions

API Docs 使用 Zola 作为静态网站生成器，并利用 GitHub Actions 在每次 Push 时自动构建与部署。

按照 官方文档 进行了相应配置，但是发现 shalzz/zola-deploy-action 使用官方仓库发布的 Zola，并不支持在构建时建立中文索引（需要额外的特性：indexing-zh），于是在其基础上创建 DeadPoetSpoon/zola-deploy-action 仓库提供该功能。
