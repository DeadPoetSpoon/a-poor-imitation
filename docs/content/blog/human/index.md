# “人”
一切思考需要从一个主体出发，API 的主体毫无疑问就是“人”，一切将由“人”或“人”的一部分组成，“人”的拆分将是本文谈论的重点。

# 脑
“人”的主体是脑，是最主要的运行机制，将其划分为：
- 行为脑：处理一切基础实体行为，包括可控与不可控部分，如呼吸、移动等。
- 感知脑：处理一切高级抽象感知，包括可控与不可控部分，如听觉、记忆等。
- 决策脑：分析信息决定一切可控行为，如决定目的地。
